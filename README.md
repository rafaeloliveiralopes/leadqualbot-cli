## üìã Project Overview

This is a **keyword-based chatbot** that runs in the terminal (CLI) to help small businesses with:
- Lead pre-screening
- Routing to: quotation, scheduling, or human support

**Stack:**
- Java 21
- Maven
- External JSON knowledge base
- No NLP, no databases, no integrations
